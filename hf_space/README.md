# TrustLens AI (Demo)

A clean, lightweight Hugging Face Space-style demo UI for **TrustLens AI**.

## What this demo shows

- **Financial product ranking** (keyword-based simulation)
- **Trust score** (0.75–0.95, formatted as a percentage)
- **LLM-style explanation** (professional, AI-like rationale)

## Run locally

```bash
cd hf_space
pip install -r requirements.txt
python app.py
```

## Note

This is a lightweight demo of TrustLens AI. Full backend includes **RAG**, **multi-LLM evaluation**, and **trust scoring**.
